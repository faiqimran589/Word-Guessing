# Word-Guessing
A console-based word guessing game developed in python


A simple and interactive Word Guessing Game built with Python. The objective is to guess the hidden word within a limited number of attempts. This project is beginner-friendly and demonstrates the use of Python fundamentals such as loops, conditionals, functions, and random word selection.

## ✨ Features
- Randomly selected words
- Limited guessing attempts
- User-friendly console interface
- Input validation
- Beginner-friendly Python project

## 🛠️ Technologies Used
- Python 3
